# Claim Assistant — Prototype

Прототип AI-ассистента оценки ущерба для клиентов СберСтрахование. Open the link with your phone, take photos, get an AI-driven damage estimate in 15 seconds.

> ⚠️ **Это прототип.** Данные не передаются в страховую компанию. Используйте для UX-тестирования и валидации AI-оценки.

---

## Быстрый старт (локально)

```bash
git clone https://github.com/<your>/claim-assistant-proto.git
cd claim-assistant-proto
npm install
cp .env.example .env.local
# Открыть .env.local и вписать ANTHROPIC_API_KEY и ADMIN_PASSWORD
npm run dev
# открыть http://localhost:3000 — но камера локально работать не будет
```

Камера в браузере требует HTTPS. Для теста на телефоне — деплоим на Vercel.

## Деплой на Vercel (3 минуты)

1. Запушить репозиторий в GitHub
2. На [vercel.com](https://vercel.com) → New Project → импортировать репозиторий
3. Environment Variables:
   - `ANTHROPIC_API_KEY` — получить на [console.anthropic.com](https://console.anthropic.com)
   - `ADMIN_PASSWORD` — любая строка для входа в `/admin`
4. Storage → Create → KV → Connect to Project (автоматически добавит `KV_*` переменные)
5. Deploy

Получите URL вида `https://claim-assistant-proto.vercel.app` — открывайте с телефона, дайте разрешение на камеру.

## Стоимость

- **Vercel Hobby:** $0 (бесплатный план хватает)
- **Vercel KV:** $0 (30K запросов/мес бесплатно)
- **Anthropic API:** ~$0.05–0.10 за один кейс (5–10 фото). На 50 тестовых кейсов — около **$3–5**.

См. раздел 9 в `docs/PRD_Prototype.md` — план эксперимента с 5–10 знакомыми, гипотезы, метрики.

## Кабинет администратора

Открыть `/admin`, ввести пароль из `ADMIN_PASSWORD`. Доступно:

- **Дашборд** — метрики по всем кейсам
- **Калибровка** — слайдеры для 11 весов (диапазон оценки, уровни отделки, пороги)
- **Справочники** — таблицы работ и материалов с импортом/экспортом XLSX
- **История** — список всех кейсов с фильтрами и экспортом в CSV

## Известные ограничения

- **iOS Safari** вырезает GPS из фото, снятых через `getUserMedia`. Координаты будут только из галереи.
- **AR-рулетка** не работает на iOS Safari (WebXR не поддерживается). Используем монету как масштаб.
- Только сценарий **«залив квартиры»** имеет полный AI-расчёт. Остальные типы событий собирают данные и переходят на «эксперт рассмотрит ваш случай».
