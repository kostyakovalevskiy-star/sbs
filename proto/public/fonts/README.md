# Fonts

Проект ожидает два шрифта из дизайн-системы Сбера:

| Семейство        | Файлы (.woff2), положить в этот каталог                                      |
|------------------|------------------------------------------------------------------------------|
| SB Sans Text     | `SBSansText-Regular.woff2`, `SBSansText-Medium.woff2`, `SBSansText-Bold.woff2` |
| SB Sans Display  | `SBSansDisplay-Regular.woff2`, `SBSansDisplay-Bold.woff2`                    |

Где взять: https://design.sber.ru (раздел «Бренд → Шрифты» или «Fonts»).
Лицензия — на стороне Сбера; уточните условия использования для вашего проекта.

Пока файлы не подложены, автоматически применяется fallback — Inter + system sans-serif. Визуально очень близко.
